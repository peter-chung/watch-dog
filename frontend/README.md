This is the Next.js frontend for Watchdog.

## Getting Started

Run the development server from the repo root:

```bash
npm run dev
```

Or run it directly from this folder:

```bash
npm run dev
```

## Environment

Required:

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`

Optional demo login:

- `DEMO_USER_EMAIL`
- `DEMO_USER_PASSWORD`
- `NEXT_PUBLIC_DEMO_USER_EMAIL`
